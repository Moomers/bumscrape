Feb 8, 2012

I've added a spider for the ticket-selling website ticketsmore. This is kind of a shady site, and all of their data is actually coming from tickettransaction.com. If you hit that url, you're taken to some sort of login form without any identifying details. I think we've stumbled upon the lair of the professional scalper!

My opinion of this was further re-enforced when I googled around and found ticket liquidator. As you can see, their listings look almost identical to the listings we're already scraping from ticketsnow, except with different prices. This site is also backed by tickettransaction.com.

Given that the JS request which retrieves ticket data contains a site ID More googling also revealed superb tickets, ticketsnow, ticketnest.com, vivid seats, and excite.com. All of these sites seem to be selling the same tickets!

BTW, seatgeek has some analysis of the 2012 lottery, although sometimes they appear to be talking about 2011.

--Igor
